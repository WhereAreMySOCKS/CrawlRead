# CrawlRead

一个基于FastAPI的Web应用程序。

## 安装

1. 克隆仓库：
```bash
git clone <repository-url>
cd CrawlRead
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 运行应用

```bash
python main.py
```

应用将在 http://localhost:8000 上运行。

## API文档

启动应用后，可以访问以下URL查看API文档：

- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
